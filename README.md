WePeiYang_iOS
=============
## 简介

微北洋 iOS 版客户端基于 Cocoa Touch 框架，以 Objective-C 与 Swift 语言编写，支持 iOS 7.0 版本以上的设备。以 Universal 模式编写，但上架版只支持 iPhone。

## 功能

* 自习室查询
* 新闻
* 公告
* GPA查询
* 图书馆
* 就业资讯
* 失物招领
